### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r requirements.txt
   $ pip install sentence-transformers
   $ pip install openpyxl
   $ pip install --upgrade openai
   ```

2. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```
